EDGE Data Project – Part 2: Analysis

This section explores how EDGE’s incentive programs shape economic development across Memphis. Using five datasets provided by EDGE, I analyzed investment patterns, job creation, and geographic distribution to understand how these programs support businesses and communities.

Guiding Questions

The analysis answers three core questions:

  • How much investment has EDGE supported across Memphis, and how does it vary by program?
  • How much did each program contribute to total jobs, which program contributed the most?
  • How is EDGE investment distributed across Memphis ZIP codes or neighborhoods?
